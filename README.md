# site-de-monitora

## 🚀 Como Executar

1. Clone o repositório ou baixe os arquivos
2. Coloque todos os arquivos na mesma pasta mantendo a estrutura
3. Abra o arquivo `index.html` no seu navegador

**Não é necessário instalar nada.**

### Credenciais de Teste
- **E-mail**: qualquer e-mail (ex: aluno@cemcs.edu.br)

## 🎨 Design

- Paleta institucional em tons de azul e verde
- Interface clean e moderna
- Totalmente responsivo (Mobile, Tablet e Desktop)
- Efeitos suaves de hover e transições

## 📱 Responsividade

O sistema foi desenvolvido seguindo a abordagem **Mobile-First** e se adapta perfeitamente a diferentes tamanhos de tela.
